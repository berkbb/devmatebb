# Changelog

### Added

- **Initial Release:** Launched `DevmateBB`, a modern dark VS Code theme.
- **Material 3 Design:** Fully built using the official Android Material 3 dark color palette.
- **Variants:** Includes standard, vibrant, and soft background theme options.
